자료형

Numbers

  • 정수형 : int
  • 실수형(소수) : float
  • 복소수 : complex (2+3j)

Boolean

  • True/False
  • 0을 제외한 숫자 True, 문자 True

String

  • 문자열
  • “, ‘ 구분없이 사용
  • “”” 세 개 사용이 멀티라인
  • 대문자/소문자 변환
    • a.upper()
    • a.lower()
  • 해당 값의 위치 찾기
    • a.find(“abc”) : 없을 경우 -1 반환
    • a.index(“a”) : 없을 경우 error
  • 포함된 개수 리턴
    • a.count(“a”)
  • 공백 제거
    • a.lstrip()
    • a.rstrip()
    • a.strip()
  • 문자열 변환
    • a.replace(“a”, “b”)
    • a.replace(“ “, “”)
  • string format

List

  • iterable
  • 여러 자료형 함께 저장 가능
  • 수열 리스트
    • b = list(range(10))
    • b = list(range(3, 10, 2))
  • 자료 수정
    • b[2] = 100
  • 자료 추가
    • b.append(“plus”)
    • b.insert(0, “insert”)
  • 자료 삭제
    • b.pop()
    • b.remove()
    • del b[2]
  • 졍렬
    • b.sort()
    • b.sort(key=len)
    • b.reverse()

Tuple

  • iterable
  • 데이터 수정 불가능
  • 리스트보다 적은 메모리 사용

Dictionary

  • key(str,int만 가능): value
  • 호출
    • d.keys()
    • d.values()
    • d.items()
    • d.get(“key1”)
  • 자료 추가
    • d[“key3”] = 40
  • 자료 삭제
    • del d[“key3”]

Set

  • 중복 데이터 없음
  • 집합의 연산 가능
  • 수정 불가능(리스트 형변환)

데이터 타입 변경

  • 문자/숫자 변환

  • boolean형 변환

  • 문자열/리스트 변환

  • 튜플/딕셔너리 변환
