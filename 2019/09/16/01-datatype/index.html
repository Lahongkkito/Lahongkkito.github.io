종류


Numbers

  • 정수형 : int
  • 실수형(소수) : float
  • 복소수 : complex (2+3j)

Boolean

  • True/False
  • 0을 제외한 숫자 True, 문자 True

String

  • 문자열

  • ", ' 구분없이 사용

  • """, ''' 세 개 사용으로 멀티라인

  • escape string

    • \n: new line
    • \t: tap

  • 대문자/소문자 변환 함수

  • 해당 값의 위치 찾기

  • 포함된 개수 리턴

  • 공백 제거

  • 문자열 변환

  • 체이닝

  • string format


List

  • iterable

  • 여러 자료형 함께 저장 가능

  • 수열 리스트

  • 자료 수정

  • 자료 추가

  • 자료 삭제

  • 졍렬

    • ls.sort() # 동일한 타입의 경우만 가능
    • ls.sort(key=len)
    • ls.reverse()

  • offset 인덱스/슬라이싱

  • 복사

    • 얕은복사 : 주소값만 복사
    • 깊은복사 : 메모리 복사

5) Tuple

  • iterable
  • 데이터 수정 불가능
  • 리스트보다 적은 메모리 사용

6) Dictionary

  • 데이터 순서 없음

  • key

    • 내부적으로 hash값으로 지정
    • str,int만 가능

  • key : value
    {key1:value2, key2:value2}

  • 호출

  • 자료 추가

  • 자료 삭제

  • update (덮어쓰기)


7) Set

  • 중복 데이터 없음

  • 교집합, 합집합, 차집합 연산 가능

  • 수정 불가능(리스트 형변환)

  • 합집합

    • a_set | b_set
    • a_set.union(b_set)

  • 교집합

    • a_set & b_set
    • a_set.intersection(b_set)

  • 차집합

    • a_set - b_set
    • a_set.difference(b_set)

  • 대칭차집합

    • a_set ^ b_set

  • 부분집합

    • a_set.issubset(b_set) = False
